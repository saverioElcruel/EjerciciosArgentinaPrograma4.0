Ejercicios Extras JS
1. Escribir una función que reciba un número entero como parámetro y devuelva la suma de todos los números pares desde 1 hasta el número ingresado.

2. Realizar un programa que solicite al usuario ingresar una lista de números separados por comas y luego muestre por pantalla el mayor y el menor número de la lista.

3. Escribir un programa que solicite al usuario ingresar una frase y luego muestre por pantalla cuántas vocales contiene la frase.

4. Crear una función que reciba un arreglo de números y devuelva la suma de todos los elementos del arreglo.

5. Realizar un programa que, dado un número entero ingresado por el usuario, muestre todos los números primos menores o iguales a ese número.

6. Escribir una función que reciba un número entero y devuelva true si es un número primo, o false en caso contrario.

7. Realizar un programa que muestre por pantalla los primeros 10 números de la serie de Fibonacci.

8. Escribir una función que reciba un arreglo de números y devuelva un nuevo arreglo con los elementos ordenados de forma ascendente.

9. Crear un programa que solicite al usuario ingresar una lista de palabras separadas por espacios y luego muestre por pantalla las palabras que tienen una longitud mayor a 5 caracteres.

10. Escribir una función que reciba un arreglo de números y devuelva un nuevo arreglo con los elementos elevados al cuadrado.

11. Realizar un programa que solicite al usuario ingresar un número entero y luego muestre por pantalla todos los números primos menores a ese número.

12. Escribir una función que reciba dos arreglos de números y devuelva un nuevo arreglo con los elementos que se encuentran en ambos arreglos.

13. Crear un programa que solicite al usuario ingresar una lista de números separados por espacios y luego muestre por pantalla el promedio de esos números.

14. Escribir una función que reciba un número entero y devuelva true si es un número palíndromo (se lee igual de izquierda a derecha que de derecha a izquierda), o false en caso contrario.

15. Realizar un programa que solicite al usuario ingresar una frase y luego muestre por pantalla la misma frase pero con todas las palabras en mayúsculas.

16. Escribir una función que reciba por parámetro un string y devuelva el mismo string con las palabras invertidas. Por ejemplo, si se utiliza el string "Hola mundo", la función deberá devolver "aloh odnum". 
Escribir una función que tome dos números como parámetros y devuelva un objeto con las operaciones matemáticas realizadas entre ellos (suma, resta, multiplicación y división). 
Crear un arreglo de objetos, transformar esos objetos en mensajes de presentación utilizando una función y mostrar los mensajes resultantes por consola
Crear una lista numerada con elementos <li> que contengan botones "subir" y "bajar", y un <span> con el nombre de un país.

17. Simular que están creando una página web de recetas y definir 5 opciones principales que deberían estar en la barra de navegación de esta página.
Comparar y analizar con el equipo si hay similitudes o diferencias en las opciones propuestas. No hay respuestas correctas o incorrectas, pero es importante considerar la experiencia del usuario y la relevancia de las secciones en la barra de navegación.
Realizar una búsqueda en el buscador de "Google" utilizando el término "página web de recetas". Visitar al menos 5 páginas de recetas analizando sus barras de navegación.
Como grupo, compartir y analizar las diferentes estructuras de las barras de navegación encontradas; ¿Coinciden con las opciones que se habían definido inicialmente? ¿Se encuentran elementos comunes entre las diferentes páginas de recetas?
